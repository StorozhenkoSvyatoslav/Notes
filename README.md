# Notes

A simple application for creating and viewing notes, developed for Android.

## Features

*   **View notes list**: The main screen displays all created notes in a staggered grid layout.
*   **Create and edit**: Users can create new notes and edit existing ones.
*   **Local storage**: Notes are saved in a local database on the device using Room, which provides access to them even offline.
*   **Navigation**: Simple navigation is implemented between the notes list and the editing screen.

## Technologies

*   **Kotlin**: The main programming language.
*   **Jetpack Compose**: A modern toolkit for building native user interfaces.
*   **MVVM (Model-View-ViewModel)**: An architectural pattern used to separate presentation logic from business logic.
*   **Room**: A library for working with the local SQLite database.
*   **Compose Navigation**: For navigating between screens in an application written in Jetpack Compose.
*   **KSP (Kotlin Symbol Processing)**: For processing annotations in Kotlin.
